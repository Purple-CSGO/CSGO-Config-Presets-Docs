# {{ $frontmatter.title }}

CFG预设有一些控制台提示方便提示玩家使用，但是常常被其他日志淹没，这里会同步最新版的各个文件提示：

## auto.cfg

```sh
═════════════════════════════════════════════════════════════
──── 加载设置[auto.cfg]  		:  exec auto 或 auto
──── 跑图,练习道具    			:  exec practice 或 pt
──── 对枪,SOLO       			:  exec solo 或 solo
──── 观战,观看DEMO,GOTV    		:  exec demo 或 demo
──── 地图中加载设置[auto.cfg]  	:  Insert键
──── 地图中一键跑图[practice.cfg]	:  Delete键
═════════════════════════════════════════════════════════════
```
